## 做出的修改

### asmhead.nas

在 `画面モードを設定` 完毕之后，`通过BIOS获取指示灯状态` 之前添加下面的代码段，使屏幕一半变为白色。

```assembly
;DS = 0xa000, SI=0, DS : SI=0xa0000
MOV BX,DS
MOV AX,0xa000
MOV DS,AX
MOV SI,0
draw:
MOV [SI],BYTE 15;15 表示白色
ADD SI,1
CMP SI,320*100;屏幕是 320*200 这里取其一半
JBE draw
MOV DS,BX
```

#### make.bat

在最后一行添加：

```
..\z_tools\make.exe run
```

使其能够在 make 之后自动运行 .`.\z_tools` 中的`make.exe`
